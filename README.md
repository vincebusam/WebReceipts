### Installation

 - Requires ```convert``` from ImageMagick

```
mkdir data
chown www-data.www-data data
```

Get an ABBYY Account at http://ocrsdk.com/, then create config.txt with

```
ABBYY_APPID=xxxx
ABBYY_PWD=yyyy
```
